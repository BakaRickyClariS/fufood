# FuFood 產品體驗流程規劃書 (Test Drive Flow)

## 角色情境

**目標角色**：主要採買者 / 冰箱管理者

---

## 1. 登入與建立家庭群組

- **快速登入**：點擊首頁「LINE 一鍵登入」快速完成帳號登入。
- **建立群組**：進入「我的冰箱」→ 點擊「建立群組」→ 命名群組（例如：`fufu home`）。
- **邀請家人**：點擊「分享邀請 QR code」→ 掃描或分享連結完成家人邀請。
- **完善帳號**：成功邀請家人後 → 返回首頁 → 點擊頭貼註冊/綁定會員。

## 2. 個人頁與會員中心

- **基本設定**：點擊個人檔案進行「註冊會員」與「角色圖像變更」。
- **飲食喜好**：設定個人/家庭「飲食喜好」，作為後續 AI 推薦食譜的依據。
- **方案升級**：查看「會員方案」→ 選擇「Pro 家庭方案」或「Free 小資入門」→ 進入結帳購物流程。
- **消耗分析**：查看「消耗原因週期報告」，掌握家庭食材浪費與使用數據。
- **客服支援**：瀏覽「其他客服問題＆幫助」。

## 3. 體驗 AI 智慧掃描入庫

- **啟動相機**：點擊下方導覽列的主按鈕 → 選擇「拍照辨識」。
- **授權與拍攝**：允許相機權限 → 拍攝或上傳含有食材的照片（例如：一瓶牛奶與一袋青菜）。
- **AI 辨識與校正**：等待 AI 辨識結果 → 核對系統自動帶出的「品項名稱」、「預估效期」與「建議備註」。
- **確認入庫**：確認資訊無誤後 → 點擊「確認入庫」或「繼續掃描辨識」以進行批次處理。

## 4. 庫存警報與靈感救援

- **庫存查看**：進入「庫存頁」，檢視剛剛入庫的食材。
- **效期警示**：觀察效期標籤的視覺顏色變化（例如：即期品自動顯示為黃色/紅色警示色）。
- **靈感救援**：點擊任一項即期食材（例如：牛奶）→ 點擊「✨ 尋找食譜靈感」。
- **食譜推薦**：觀看系統根據所選食材，AI 生成的專屬食譜（例如：白醬義大利麵）。

## 5. 採買同步與即時通知 (食譜聯動)

- **消耗與補貨**：在食譜頁面底部 → 點擊「消耗食材」，並選擇將缺少的食材加入清單（例如：自動加入還缺少的洋蔥、培根）。
- **清單同步**：切換至「規劃中心」的「待採買清單」分頁，確認剛才缺少的食材已自動加入清單。
- **通知提醒**：切換至「通知頁」，確認系統已推播「已有食材消耗」之相關提醒。

## 6. 即時通知冰箱大小事

- **通知總覽**：在「通知頁」中集中查看 3 大類別即時動態：
  - **庫存狀態**：即期提醒、已消耗通知。
  - **採買狀態**：共享清單的新增/修改/勾選完成狀態。
  - **群組動態**：好友或家人成功加入群組的通知。
